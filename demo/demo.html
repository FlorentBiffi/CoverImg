jQuery.CoverImg - fullscreen demo

Note: This fullscreen demo works with a "img" html tag but can also works with a background-image on the container. Scroll for see the others demos and resize your window for see the result. 

$(".container-img").coverImg();

jQuery.CoverImg - responsive demo

Note: The container have a fixed size. This size change in function of size of window (responsive). In this demo jQuery.CoverImg works with a "img" html tag. Smartresize option active the debouncing function from John Hann.

$(".container-img1").coverImg(
{
	'fullscreen': false,
	'smartresize' : true //debouncing function from John Hann
});

Note: jQuery.CoverImg width fixed container size and background image on container. 

$(".container-img2").coverImg(
{
	'type': 'background',
	'fullscreen': false,
	'imgWidth' : 1920, //this value is equal at the width of image
	'imgHeight' : 1200 //this value is equal at the height of image
});

Warning: When 'type' option is 'background', the 'imgWidth' and 'imgHeight' options are mandatory !
